# Unit 6 – CI/CD with Jenkins ⚙️🚀🐳

This branch contains the practical work, notes, screenshots, and documentation related to **Unit 6: Continuous Integration and Continuous Deployment (CI/CD) using Jenkins**.

The focus of this unit is to understand Jenkins architecture, pipeline creation, integration with GitHub, Docker, and Maven, and deployment automation.

Jenkins is one of the most widely used tools in DevOps for automating build, test, and deployment workflows.

- Jenkins installation  
- Jenkins UI overview  
- Plugin management  
- Jenkins security setup  
- User and role management

## 🔧 Jenkins Pipelines

- Freestyle jobs vs Pipeline jobs

- Pipeline parameters
