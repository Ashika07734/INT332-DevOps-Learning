# 🛠️ Installation Guide – Maven & Docker Setup

This document provides the steps required to install all necessary tools for running the Unit-4 Maven and Docker project.

---

# 📋 Prerequisites

Make sure the following software is installed on your system:

- Java JDK 17 or above  
- Apache Maven  
- Docker Desktop  
- Git  
- Code Editor (VS Code / IntelliJ IDEA recommended)

---

# ⚙️ Step 1 – Install Java (JDK)

Check if Java is installed:

```bash
java -version
```
If Java is not installed:

- Download JDK 17 or above
- Install Java
- Set JAVA_HOME environment variable
- Add Java to system PATH

  ---
# ⚙️ Step 2 – Apache Maven
Check Maven installation:
```bash
mvn -version
```
If Maven is not installed:

- Download Apache Maven
- Extract the Maven folder
- Set MAVEN_HOME environment variable
- Add Maven bin folder to PATH

---

- Ensures a **stable development environment**

A correct installation setup helps developers avoid runtime errors and ensures smooth execution of Maven and Docker commands.
